Welcome to dust-motes

dust is a JavaScript templating engine. It comes with a number of standard helpers. Adding every helper anyone might need to the standard set would get crazy and overload the commonly used set. This repository is a collecting point for contributed dust helpers and dust filters that provide features not supported by the standard set.
